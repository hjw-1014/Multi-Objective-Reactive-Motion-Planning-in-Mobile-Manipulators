#####################################################################################
----------------------- iiwas Description Package ----------------------------------
#####################################################################################

This package contains the urdf model of iiwas in robots/iiwas.urdf.xacro .
Moreover you can find a urdf model without hands (just kuka arms) in robots/iiwas_no_hands.urdf.xacro

To visualize the urdf model in rviz (with additional fake joint controllers) you can use
roslaunch iiwas_description iiwas_rviz.launch (currently not working under kinetic because o strange EOF error)
