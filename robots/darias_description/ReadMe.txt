#####################################################################################
----------------------- Darias Description Package ----------------------------------
#####################################################################################

This package contains the urdf model of darias in robots/darias.urdf.xacro .
Moreover you can find a urdf model without hands (just kuka arms) in robots/darias_no_hands.urdf.xacro

To visualize the urdf model in rviz (with additional fake joint controllers) you can use
roslaunch darias_description darias_rviz.launch (currently not working under kinetic because o strange EOF error)
